# Documentation Standards

This folder contains **rules and standards** for writing documentation in this project. Use them when creating or reviewing docs.

| Document | Purpose |
|----------|---------|
| [Tool documentation rules](tool-documentation-rules.md) | How to document tools (APIs, MCP tools, CLI, library functions): function behavior, inputs, parameters, outputs, examples. |
| [Dataset codebook and README rules](dataset-codebook-readme-rules.md) | How to write dataset READMEs and codebooks: file and variable descriptions, background for collection and use. |

Apply these rules so documentation is consistent, complete, and easy to follow.
